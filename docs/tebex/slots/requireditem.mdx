---
title: "Required Admin"
---

command

```lua
lib.addCommand('openManagement', {
    help = 'Open management y papi (all perms)',
    params = {
        {
            name = 'target',
            type = 'playerId',
            help = 'Target player\'s server id',
            optional = true,
        },
    },
    restricted = 'group.admin'
}, function(source, args, raw)
    local targetId = args.target or source
    if targetId then
        local perms = {'rules', 'admin', 'admin:setCrew', 'admin:removeCrew', 'admin:createCrew'}
        lib.triggerClientEvent('papi_management:client:openDashboard', targetId, perms)
    end
end)
```

exports use 

```lua
local perms = {'rules', 'admin', 'admin:setCrew', 'admin:removeCrew', 'admin:createCrew'}
exports.papi_management:openDashboard(source, perms)
```

not advisable (required config.adminExternal)
